# Sistema de Controle de Vendas (Java)

## 📌 Descrição
Projeto desenvolvido em Java com o objetivo de simular um sistema de vendas em ambiente console, aplicando conceitos de lógica de programação e controle de fluxo.

## 🚀 Funcionalidades
- Seleção de produtos
- Cálculo automático do valor total da compra
- Menu interativo no terminal
- Exibição do total com formatação

## 🛠️ Tecnologias utilizadas
- Java
- Estruturas condicionais (if/else)
- Estruturas de repetição (do-while)

## 🎯 Objetivo do projeto
Este projeto foi desenvolvido como parte dos estudos em programação, com foco na prática de lógica de negócio e manipulação de dados.

## ▶️ Como executar
1. Compile o arquivo `.java`
2. Execute a classe `SistemaVendas`
3. Utilize o menu para simular uma compra
